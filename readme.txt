BusinessPay is a Ghanaian Payment Gateway Add-on for the GiveWP plugin.

== Description ==

This is the BusinessPay payment gateway for GiveWP. Allows you to use Asoriba BusinessPay payment gateway for the GiveWP plugin. BusinessPay is a Ghanaian Payment Gateway.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.
1. Ensure you have latest version of GiveWP plugin installed
2. Unzip and upload file & folder of the plugin to your /wp-content/plugins/ directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Make hover on Donations menu and click on Settings menu from left menu
5. Click on Payment Gateways tab and enable Asoriba BusinessPay payment gateway
6. Configure all required settings and see the result on the donation form

== Configuration ==
1. Visit the Donations settings page, and click on the Payment Gateways tab.
2. Click on BusinessPay to edit the settings. If you do not see BusinessPay in the list at the top of the screen make sure you have activated the plugin in the WordPress Plugin Manager.
3. Enable the Payment Method, name it BusinessPay, add in your public key and sandbox enable/disable Click Save.
